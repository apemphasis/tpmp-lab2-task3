# Отчет о тестировании программы Marshrut

В данном документе представлены результаты функционального тестирования консольного приложения для работы с маршрутами.

|  | Входные данные | Ожидаемые выходные данные | Действительные выходные данные | Статус |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Выбор пункта меню `2` при `defined == false` | Сообщение: `Маршруты не заданы` | `Маршруты не заданы` | Пройден |
| 2 | Пункт `1`, ввод количества: `15` (больше `MAX_MARSHUTS`) | Ошибка: `Количество должно быть от 1 до 10.` | `Ошибка! Количество должно быть от 1 до 10.` | Пройден |
| 3 | Пункт `1`, ввод количества: `2`. Данные: <br>1. 15, Минск, Брест, 350.5<br>2. 5, Гомель, Минск, 310.0 | Подтверждение: `Создано 2 маршрутов.` | `Создано 2 маршрутов.` | Пройден |
| 4 | Пункт `3` (поиск максимальной длины) | Вывод данных маршрута 15 (длина 350.50) | Вывод маршрута 15 | Пройден |
| 5 | Пункт `4` (сортировка по номерам) | Сообщение о завершении сортировки | `Маршруты отсортированы по возрастанию номеров.` | Пройден |
| 6 | Пункт `2` (просмотр списка после сортировки) | Список: сначала маршрут 5, затем 15 | Список отображен в порядке: 5, 15 | Пройден |
| 7 | Пункт `5`, поиск по строке: `Минск` | Найдено 2 маршрута (Брест-Минск и Гомель-Минск) | Найдено 2 маршрута | Пройден |
| 8 | Пункт `5`, поиск по строке: `Гродно` | Сообщение: маршруты не найдены | `Маршруты, начинающиеся или заканчивающиеся в пункте 'Гродно', не найдены.` | Пройден |
| 9 | Пункт `0` (выход из программы) | Завершение процесса выполнения | `Программа завершена.` | Пройден |
