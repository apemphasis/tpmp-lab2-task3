# Инструкция по установке Marshrut Project

# Требования
- Операционная система: Linux/Unix
- Компилятор: GCC
- Утилита: make

# Шаги установки

1. Распакуйте архив с проектом:
   tar -xzf marshrut_project.tar.gz

1. Перейдите в директорию проекта:
   cd marshrut_project

1. Соберите проект:
   make all

1. Запустите программу:
   ./bin/marshrut

# Структура директорий после сборки

- bin/ - исполняемый файл
- obj/ - объектные файлы
- include/ - заголовочные файлы
- src/ - исходные коды
- docs/ - документация
