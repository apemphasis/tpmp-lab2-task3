# Marshrut Project

## Overview
Проект представляет собой консольное приложение на языке C для работы с данными о маршрутах. Программа позволяет создавать и управлять массивом структур `MARSHRUT`, каждая из которых содержит номер маршрута, начальный пункт, конечный пункт и длину маршрута. Максимальное количество маршрутов в системе ограничено 10, а максимальная длина названия пункта составляет 50 символов. 

Пользователю предоставляется интерактивное меню, с помощью которого можно:
* Создавать массив маршрутов путем ввода данных с клавиатуры.
* Выводить на экран список всех сохраненных маршрутов.
* Находить и отображать маршрут с максимальной длиной.
* Сортировать массив маршрутов по возрастанию их номеров.
* Выполнять поиск маршрутов, которые начинаются или заканчиваются в заданном пункте.

## Usage
Для успешной компиляции и работы с проектом требуются операционная система Linux/Unix, компилятор GCC и утилита `make`.

**Сборка проекта:**
Перейдите в корневую директорию проекта и выполните команду сборки:
```bash
make all
```
Также поддерживается альтернативная команда для сборки: `make build`.

**Запуск программы:**
После успешной сборки запустить исполняемый файл можно командой:
```bash
./bin/marshrut
```
Вы также можете скомпилировать и сразу запустить программу одной командой:
```bash
make run
```

**Очистка директории:**
Для удаления сгенерированных объектных файлов и исполняемого файла используйте команду:
```bash
make clean
```

## Additional Notes
* **Структура директорий:** После сборки в проекте будут присутствовать следующие директории: `bin/` (содержит исполняемый файл), `obj/` (содержит скомпилированные объектные файлы), `include/` (заголовочные файлы), `src/` (файлы исходного кода) и `docs/` (документация).
* **Безопасность меню:** Если массив маршрутов не был создан (пункт 1 в меню), попытки вывести маршруты, отсортировать их или выполнить поиск вызовут предупреждающее сообщение о том, что массив пуст.
* **Ограничения ввода:** При создании массива программа запрашивает количество маршрутов и принимает только корректные значения (от 1 до 10 включительно), в противном случае выводится сообщение об ошибке.
